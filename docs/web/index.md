# Data table
Deve receber valores true ou false para exibir os botoes:
- Update
- Delete
- Create

## Formatação especial

Para datas use a formatação DD/MM/YYYY. Exemplo: 04/05/2026. 
Para status utilize um icone(shield-check, shield-ban) do lado do texto e dentro de um badge.
Exemplo:
- 🛡️ Ativo
- 🛡️ Inativo
